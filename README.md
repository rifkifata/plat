## Brace Your Self

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rifkifata/plate.git
   ```
2. Install NPM packages
   ```sh
   npm install --save
   ```
3. Start The Server
   ```sh
   node ./index.js
   ```
